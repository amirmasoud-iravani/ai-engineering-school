# Learning Roadmap

## Stage 0 — Setup

Learn folders, files, Jupyter, VS Code, virtual environments, Git, and GitHub.

## Stage 1 — Python

Learn variables, core data types, operators, conditions, loops, functions, files, exceptions, modules, and beginner coding habits.

## Stage 2 — NumPy

Learn arrays, shapes, indexing, vectorized operations, broadcasting, random numbers, and basic linear algebra.

## Stage 3 — pandas

Learn Series, DataFrames, loading data, inspecting rows, selecting columns, cleaning values, grouping, joining, and exporting results.

## Stage 4 — Matplotlib

Learn figures, axes, line plots, bar plots, scatter plots, histograms, labels, legends, and saving plots.

## Stage 5 — Data preprocessing

Learn missing values, categorical encoding, scaling, train/validation/test splits, feature engineering, pipelines, and data leakage.

## Stage 6 — Machine learning

Learn regression, classification, metrics, baselines, trees, ensembles, cross-validation, hyperparameter search, and error analysis.

## Stage 7 — Deep learning

Learn tensors, automatic differentiation, neural-network modules, training loops, optimizers, regularization, CNNs, sequence models, transformers, and GPU use.

## Stage 8 — MLOps

Learn reproducible environments, tests, experiment tracking, model packaging, APIs, containers, CI/CD, logging, monitoring, and model/version management.

## Stage 9 — AI applications

Build usable applications: model APIs, Streamlit demos, retrieval-augmented generation, tool-using agents, and evaluation pipelines.

## Stage 10 — Portfolio projects

Build complete projects with a problem statement, data card, experiments, model card, tests, deployment, monitoring plan, and clear README.
