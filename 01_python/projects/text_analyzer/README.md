# Mini-project: Persian-friendly text analyzer

This small project combines strings, lists, dictionaries, sets, functions, files, exceptions, modules, and operators.

## Run it

From the repository root:

```bash
python 01_python/projects/text_analyzer/text_analyzer.py \
  01_python/projects/text_analyzer/sample_text.txt
```

On Windows PowerShell, the command may be written on one line.

## What it reports

- number of characters;
- number of whitespace-separated tokens;
- number of unique normalized tokens;
- most frequent tokens.

This is deliberately a beginner project, not a linguistically complete Persian tokenizer. Later NLP lessons can replace the simple tokenization with proper normalization and tokenization.
