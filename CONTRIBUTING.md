# Contributing to your own school

When adding a lesson:

1. give it a numerical name;
2. begin with a simple idea;
3. include small runnable examples;
4. add one exercise;
5. avoid unexplained magic;
6. test the notebook from top to bottom;
7. make a clear Git commit.

Useful commit messages:

```text
Add NumPy array basics
Fix list sorting explanation
Add pandas cleaning exercises
```
